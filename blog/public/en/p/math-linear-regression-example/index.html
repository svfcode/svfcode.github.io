Example: Linear Regression

Gradient descent and chain rule on a simple model

Mar 13, 2026
3 minute read
ru

Linear regression \( \hat{y} = wx + b \) — a minimal model ideal for understanding gradient descent, the chain rule, and how derivatives drive weight updates.

Assumes familiarity with the gradient and chain rule.

Model and loss

Model: \( \hat{y} = wx + b \)

Loss — mean squared error (MSE) for one example:

\[ L = \frac{1}{2}(y - \hat{y})^2 \]

The \( \frac{1}{2} \) factor is for convenience when differentiating (it cancels with the 2 from the square).

Derivatives via the chain rule

We need \( \frac{\partial L}{\partial w} \) and \( \frac{\partial L}{\partial b} \).

Chain: \( w, b, x \xrightarrow{} \hat{y} \xrightarrow{} L \)

\[ \frac{\partial L}{\partial w} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial w} \]\[ \frac{\partial L}{\partial b} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial b} \]

Computing:

  • \( \frac{\partial L}{\partial \hat{y}} = \frac{\partial}{\partial \hat{y}}\left[\frac{1}{2}(y - \hat{y})^2\right] = -(y - \hat{y}) \)
  • \( \frac{\partial \hat{y}}{\partial w} = x \)
  • \( \frac{\partial \hat{y}}{\partial b} = 1 \)

Result:

\[ \frac{\partial L}{\partial w} = -(y - \hat{y}) \cdot x \]\[ \frac{\partial L}{\partial b} = -(y - \hat{y}) \]

Intuition

  • Error \( (y - \hat{y}) \): the larger the mistake, the more we correct
  • Weight \( w \): the gradient includes \( x \) — if input \( x \) is large, this input’s contribution to the error is larger, so the update to \( w \) is larger
  • Bias \( b \): the gradient doesn’t depend on \( x \) — \( b \) shifts the prediction uniformly

Weight updates (gradient descent)

\[ w_{new} = w_{old} - \alpha \cdot \frac{\partial L}{\partial w} = w_{old} + \alpha (y - \hat{y}) x \]\[ b_{new} = b_{old} - \alpha \cdot \frac{\partial L}{\partial b} = b_{old} + \alpha (y - \hat{y}) \]

When \( \hat{y} < y \) (underestimation): \( (y - \hat{y}) > 0 \), \( w \) and \( b \) increase, prediction goes up. When \( \hat{y} > y \) (overestimation): \( (y - \hat{y}) < 0 \), weights decrease.

For a batch (multiple examples)

For \( n \) examples \( (x_i, y_i) \), average loss:

\[ L = \frac{1}{n} \sum_{i=1}^{n} \frac{1}{2}(y_i - \hat{y}_i)^2 \]

Gradients — averages over the batch:

\[ \frac{\partial L}{\partial w} = -\frac{1}{n} \sum_{i} (y_i - \hat{y}_i) x_i \]\[ \frac{\partial L}{\partial b} = -\frac{1}{n} \sum_{i} (y_i - \hat{y}_i) \]

Summary

  • Linear regression — simple chain \( w, b \to \hat{y} \to L \)
  • Chain rule yields \( \frac{\partial L}{\partial w} = -(y - \hat{y}) x \), \( \frac{\partial L}{\partial b} = -(y - \hat{y}) \)
  • Gradient descent updates weights in the direction that decreases loss

The same principle applies in neural networks: only the chain is longer, and PyTorch/TensorFlow compute gradients automatically.
