Чему равна производная функции g(x) = 3x − 1? (в любой точке x)
Подсказка

Для прямой y = kx + b наклон (производная) везде равен коэффициенту k.

Чему равна f'(4) для f(x) = x²?
Подсказка

Производная (x²)' = 2x. Подставь x = 4.

Машина едет 60 км/ч. s(t) = 60t. Чему равна s'(t) (мгновенная скорость)?
Подсказка

При постоянной скорости производная расстояния по времени = сама скорость.
